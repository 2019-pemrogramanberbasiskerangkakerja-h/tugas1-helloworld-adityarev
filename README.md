## How to Run

- Install client `node_modules` on `/client` directory
  ```sh
  run *npm install*
  run *npm start*
  ```
- Install server `node_modules` on `/` directory
  ```sh
  run *npm install*
  run *npm start*
  ```
- Open `localhost:3000`
